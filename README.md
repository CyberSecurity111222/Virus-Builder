```sh
git clone https://github.com/CyberSecurity111222/Virus-Builder/Virus-Builder
```

```sh
cd Virus-Builder
```
Step 3
```sh
ls
```
Step 4
```sh
pip install -r requirements.txt
```

python3 Builder.py
